Generate leads from contact form 7 submission

== Description ==
Records all the submissions from contact form 7 and saves them in database . You can check the same in custom page inside wordpress admin.

== Installation ==
ADMIN INSTALLER VIA SEARCH
Visit the Add New plugin screen and search for “CFLEAD”.
Click the “Install Now” button.
Activate the plugin.
Navigate to the “Leads” Menu.
ADMIN INSTALLER VIA ZIP
Visit the Add New plugin screen and click the “Upload Plugin” button.
Click the “Browse…” button and select zip file from your computer.
Click “Install Now” button.
Once done uploading, activate CFLEAD
MANUAL
Upload the CFLEAD folder to the plugins directory in your WordPress installation.
Activate the plugin.
Navigate to the “Leads” Menu.

== Frequently Asked Questions ==
Will it be working for any contact form?
No the current plugin is compatible only with contact form 7
